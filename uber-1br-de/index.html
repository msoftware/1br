Über 1br.de

Die Seite 1br.de ist ein Experiment, bei dem es darum geht, einen WordPress Blog bei github.com zu hosten. Dabei wird von der WordPress Seite jede Nacht ein Mirror der HTML Seiten gemacht. Die statischen Dateien werden dann in das github Repository unter https://github.com/msoftware/1br geschireben kopiert und sind dann hier unter http://www.1br.de/ zu ereichen.
